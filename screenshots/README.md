# Screenshots

This directory contains screenshots of the QuizCraft application.

## Directory Structure

- `web/` - Screenshots from the web version
- `mobile/` - Screenshots from the mobile version

## Adding Screenshots

Please add your screenshots to the appropriate directories:

### Web Screenshots
- `home.png` - Home page
- `quiz.png` - Quiz playing interface
- `results.png` - Quiz results page
- `leaderboard.png` - Leaderboard page
- `profile.png` - User profile page
- `admin.png` - Admin panel (optional)

### Mobile Screenshots
- `home.png` - Home screen
- `quiz.png` - Quiz playing screen
- `results.png` - Quiz results screen
- `profile.png` - Profile screen
- `leaderboard.png` - Leaderboard screen (optional)
