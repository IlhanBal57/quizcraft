# Assets Folder

Bu klasör uygulama için gerekli asset dosyalarını içerir.

## Gerekli Dosyalar

### Ses Dosyaları (sounds/)
- `correct.mp3` - Doğru cevap sesi
- `wrong.mp3` - Yanlış cevap sesi

### Görsel Dosyalar
- `icon.png` - Uygulama ikonu (1024x1024 px önerilen)
- `splash.png` - Splash ekranı (1242x2436 px önerilen)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)

## Notlar

- Ses dosyaları için royalty-free kaynaklardan temin edilebilir:
  - https://freesound.org/
  - https://mixkit.co/free-sound-effects/
  
- İkon ve splash için:
  - https://www.figma.com/
  - https://canva.com/

Expo'nun `expo-asset` modülü ile bu dosyalar otomatik olarak bundle edilir.
